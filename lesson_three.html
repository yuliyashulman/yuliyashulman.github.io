Why not, or the Actual Lesson

Review went okay, I showed coordinates again, putting coordinate grid on the background of the screen. This helped with visualization.

Then I explained the variables. Khan academy video confused the girls. Children were completely unprepared for the format of the information presented, and I almost lost them. We quickly regrouped and shown bucket, called it a variable, and started putting different things in it. Those were values. Bucket was a named placeholder for the values. This was more clear.

Then we started putting together a calculator project. First we created a new project. It has cat sprite by default, which we deleted. We did not pick a new sprite, instead we attached code to the canvas. We created a little bit of code to take user input and store it into variables, and outputting sum into the result. We did not go beyond summation, as there was a lot of confusion, and we had to go over concepts multiple times. It was hands down the bumpiest lesson, but I could not quite put my finger as to why. Calculator - Addition Only

Luckily, my test audience is comprised of my friends, who are not shy about giving feedback. This time it was negative. One girl described it as too technical, but I did not get a sense that she was lost during the lesson. Looking back I see that too many new concepts were introduced too fast, and the lesson was overwhelming for the young audience. I was lucky I was able to fix this mistake before it was too late, and the girl did not lose her interest.

Takeaway: don't rush. These concepts are fundamental, and the girls need time to absorb and process. Revisit next lesson format.

Recommended Lesson Plan

Dedicate the lesson to explaining the user input and create a project using graphics and interractions. If time permists, introduct loop counter.

Brag of the day: By this point, some of the girls started programming at home on their own. The ones who ventured into exploratory work were eager to share it. So I decided to slow down, turn the table, and let the girls drive the next lesson.
